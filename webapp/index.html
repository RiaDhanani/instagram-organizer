How it works

1
Install the Chrome extension Add Saved Posts Organizer from the Chrome Web Store.
2
Export from Instagram Go to instagram.com → Saved → open any collection → click the extension → Export.
3
Organize automatically The app opens with your posts ready. Click Auto Categorize — no API key needed, categorization runs on our servers automatically.
